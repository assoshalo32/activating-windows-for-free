Activating Windows 10 For Free

    Windows OS is a powerful and widely used operating system. It is a product of Microsoft and is the most used operating system worldwide. Windows comes in many versions (e.g XP, Vista, 8, 10 etc), but we'll be focusing on Windows 10 on this page.

    It is not unknown to most that Windows 10 (or any other version of windows) requires an activation to be fully utilized, and this activation is not all that cheap.

    If you're on this webpage, you are probably looking for a way to activate windows 10 for free. This will be discussed below. A few things you'll need:

  • A functioning computer running Windows 10 and is not activated.
  • Access to admin user (you'll need to run some applications as administrator).
  • A stable Internet connection.
  • If you are using the second method, you'll need to know what version of windows 10 you are on.

Method 1(recommended):

  1. Run Windows Powershell as Administrator:

    2. Open windows search and type in Powershell. Open it as administrator and wait for a few seconds.


  2. Type in the following Commands:

    3. irm https://get.activated.win | iex

    Copy and run this code on the Powershell interface.

  3. Select the First Option:

    4. After running the code, wait a few seconds as it connects. After that, you'll see a screen like this:

    Press 1 and wait a few seconds.

  4. Completion:

    5. After the wait, a text should appear indicating that Windows is activated. Exit the application and check your activation status in settings. Windows should be activated now!

    Errors may arise if connection is unstable or if server traffic is high. If no errors arise, congratulations on activating Windows!
